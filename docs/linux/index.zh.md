## 工具类别

### 文本处理

- **grep** - 模式搜索和匹配
- **sed** - 流编辑和文本转换
- **awk** - 文本处理和数据提取

### 文件管理

- **find** - 定位文件和目录
- **locate** - 快速文件定位
- **which** - 命令位置实用程序

### 系统监控

- **ps** - 进程监控
- **top/htop** - 系统资源监控
- **netstat** - 网络连接监控
