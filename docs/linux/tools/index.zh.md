## 工具类别

### 文本处理

- **grep** - 模式搜索和匹配
- **sed** - 流编辑和文本转换
- **awk** - 文本处理和数据提取
- **cut** - 从分隔文件中提取列
- **sort** - 文本排序和排列

### 文件管理

- **find** - 定位文件和目录
- **locate** - 使用数据库快速定位文件
- **which** - 命令位置实用程序
- **ls** - 具有高级选项的目录列表
- **du** - 磁盘使用分析

### 系统监控

- **ps** - 进程监控和管理
- **top/htop** - 实时系统资源监控
- **netstat** - 网络连接监控
- **iostat** - I/O 统计和性能
- **vmstat** - 虚拟内存统计
